# About

Verdict is a narrative courtroom game developed using **Unity 6** by **Ø** for a game development competition.

Players assume the role of both an investigator and a defense attorney. Every decision made during the investigation directly affects the courtroom battle, where evidence, testimony, and logical reasoning determine the final verdict.

Our goal is to create a courtroom experience that rewards **critical thinking**, **careful observation**, and **logical argumentation** instead of simple trial-and-error gameplay.

---

# Core Gameplay Loop

```text
Receive Case
      ↓
Investigate Crime Scene
      ↓
Collect Evidence
      ↓
Interview Witnesses
      ↓
Build Case Theory
      ↓
Courtroom Trial
      ↓
Cross Examination
      ↓
Present Evidence
      ↓
Reach Verdict
```

# Design Principles

The project follows several core development principles:

- Data-driven architecture
- Modular gameplay systems
- ScriptableObject-based content authoring
- Player reasoning over memorization
- Separation between data and runtime logic
- Small concept polished over large unfinished scope
